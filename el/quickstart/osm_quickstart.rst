Προγράμματα
========

JOSM
~~~~

**JOSM** (Java OpenStreetMap Editor) είναι ένας editor για το OpenStreetMap (OSM)
γραμμένος σε Java. Η τρέχουσα έκδοση υποστηρίζει αυτόνομα GPX tracks,
GPX track δεδομένα από OSM βάση δεδομένων και υπάρχοντες κόμβους,γραμμικά στοιχεία και καρτέλες μεταδεδομένων από την OSM βάση δεδομένων.

* Αρχική σελίδα: http://josm.openstreetmap.de

Η συλλογή JOSM με τα πρόσθετα στοιχεία περιλαμβάνει τα ακόλουθα πρόσθετα στοιχεία:

* agpifoj	     - Δημιουργεί ένα νέο επίπεδο με γεοαναφερμένες εικόνες.
* colorscheme	     - Επιτρέπει τη δημιουργία διαφορετικών χρωματικών συνδυασμών.
* livegps	     - Υποστήρίζει δεδομένα εισόδου από live GPS παράλληλα με δεδομένα gpsd.
* measurement	     - Προσθέτει dialog και επίπεδο για τη μέτρηση μήκους και γωνίας των στοιχείων και δημιουργεί measurement paths.
* openvisible	     - Επιτρέπει το άνοιγμα αρχείων gpx και osm τα οποία διατέμνουν ην τρέχουσα ορατή περιοχή οθόνης.
* slippymap	     - Παρουσιάζουν ένα ολισθαίνοντα κάνναβο χάρτη στο JOSM.
* surveyor	     - Επιτρέπει την προσθήκη σημαδιών/κόμβων σε τρέχουσες θέσεις gps.
* utilsplugin	     - Διάφορες βοηθητικές λειτουργίες.
* validator	     - Επαληθευτής δεδομένων.
* wmsplugin	     - Παρουσιάζει εικόνες φόντου απο οποίαδήποτε πηγή WMS.

Merkaartor
~~~~~~~~~~

**Merkaartor** είναι ένα εργαλείο σύνταξης χάρτη για το OpenStreetMap το οποίο χρησιμοποιεί το Qt toolkit.
Είναι λίγο πιο φιλικό στο χρήστη από το JOSM, αλλά έχει λιγότερα χαρακτηριστικά.

* Αρχική σελίδα: http://merkaartor.be
* Διαδικτυακή βοήθεια: http://www.merkaartor.org/Documentation/


Gosmore
~~~~~~~

Το **Gosmore**είναι ένα πρόγραμμα επισκόπησης του OpenStreetMap, εργαλείο εύρεσης διαδρομής, and search client
with support for speech synthesis and fetching the current user's
current GPS location using `gpsd <http://gpsd.berlios.de>`_.

* Αρχική σελίδα: http://wiki.openstreetmap.org/index.php/Gosmore
* Διαδικτυακή βοήθεια wiki : http://wiki.openstreetmap.org/wiki/Gosmore#User_Interface
* Πριν ρέξετε το Gosmore θα πρέπει να μετατρέψετε ένα `planet-*.osm` αρχείο δεδομένων. Κάποια δεδομένα παρέχονται στον κατάλογο `/usr/local/share/osm/` .

Μπορείτε να τα εισάγετε με:

::

  bzip2 -dc /usr/local/share/osm/Barcelona.osm.bz2 | gosmore rebuild

Αυτό θα δημιουργήσει το gosmore.pak.


Osmarender
~~~~~~~~~~

Το**Osmarender** είναι ένα is a rule-based εργαλείο φωτορεαλισμού για τη δημιουργία SVG εικόνων από δεδομένα OSM. Χρειάζεται σαν στοιχεία εισόδου ένα αρχείο δεδομένων OpenStreetMap και ένα rules file.
Εισάγει μια SVG εικόνα η οποία είναι marked up σε συμφωνία με τα στυλ που καθορίζονται στο rule file.

* Αρχική σελίδα: http://wiki.openstreetmap.org/wiki/Osmarender

Παράδειγμα:

.. Σημείωση: Αυτό απαιτεί περίπου  1 GB ελεύθερη RAM και παίρνει μερικές ημέρες για να επεξεργασθεί ολόκληρη πόλη.
   the full city. Ένα μόνο χωριό ή προάστιο θα ήταν περισσότερο διαχειρίσιμο.
   Δείτε `Xapi <http://wiki.openstreetmap.org/wiki/Xapi>`_ στο OSM Wiki για
   λεπτομέρειες σχετικά με το πώς θα κατεβάσετε μικρότερα κομμάτια σε ένα ``.osm`` αρχείο, ή πώς θα χρησιμοποιήσετε το 
   *JOSM* ή το *Merkaartor*για να κατεβάσετε μικρότερα κομμάτια.

::

  cp /usr/local/share/osm/Barcelona.osm.bz2 .
  bzip2 -d Barcelona.osm.bz2
  osmarender Barcelona.osm

Μπορείτε να δείτε τα αποτελέσματα σε ένα SVG viewer όπως ο `Inkscape` η ο `Firefox`:

::

  firefox Barcelona.svg


osm2pgsql
~~~~~~~~~

**osm2pgsql**είναι ένα λειτουργικό πρόγραμμα το οποίο μετατρέπει δεδομένα OpenStreetMap (.OSM) σε κατάλληλο τύπο ώστε να μπορούν να "¨φορτωθούν" στο (PostGIS). Χρησιμοποιείται συχνά για την οπτικοποίηση των OSM δεδομένων με χρήση του Mapnik, καθώς το Mapnik υποστηρίζει ερωτήματα προς PostgreSQL για χαρτογραφικά δεδομένα, αλλά δεν δουλεύει απ'ευθείας με αρχεία OSM.

* Αρχική σελίδα: http://wiki.openstreetmap.org/wiki/Osm2pgsql
