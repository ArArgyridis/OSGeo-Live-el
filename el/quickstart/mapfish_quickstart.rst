********************
Οδηγίες γρήγορου ξεκινήματος mapfish
********************

Το MapFish είναι ένα ευέλικτο και πλήρες πλαίσιο εργασίας για τη διαμόρφωση πλούσιων web-mapping
εφαρμογών. Δείνει έμφαση σε υψηλή παραγωγικότητα και υψηλής ποιότητας ανάπτυξη. 

Αυτός ο οδηγός γρήγορου ξεκινήματος περιγράφει πως (α) Μπορεί κάποιος να τρέξει την μικρότερη εφαρμογή MapFish που είναι εγκατεστημένη σε αυτό το live DVD, (β) Μπορούν να δημιουτγηθούν MapFish εφαρμογές, και (γ) πώς μπορεί να εγκαταστήσει τοMapFish.

Τρέξιμο της μικρότερης εφαρμογής
===========================

Για να τρέξει η μικρότερη εφαρμογή, κάνουμε κλικ στο MapFish link στο Desktop. Αυτή η κίνηση θα ξεκινήσει τον server και θα ανοίξει ένα FireFox tab που θα δείχνει την εφαρμογή
(http://localhost:5000).

Η μικρότερη εφαρμογή περιλαμβάνει ένα χάρτη με δύο `OSM
<http://www.openstreetmap.org>`_ layers, μια γραμμή εργαλείων πάνω από το χάρτη, και ένα layer
tree.

Πώς δημιουργούμε μια εφαρμογή με το MapFish
============================

Το πλαίσιο εργασίας του MapFish είναι εγκατεστημένο στο live DVD στο virtual Python περιβάλλον, το οποίο βρίσκεται στο ``/usr/lib/mapfish/mapfish-venv`` directory. Εμείς, θα χρησιμοποιήσουμε την εγκατάσταση αυτού του πλαισίου εργασίας εδώ.</p>

Μια εφαρμογή MapFish δημιουργείται με τρεις εντολές (από ένα τερματικό):

* Ενεργοποιούμε το περιβάλλον virtual Python:

    $ source /usr/lib/mapfish/mapfish-venv/bin/activate

* Δημιουργούμε την εφαρμογή:

    $ paster create -t mapfish MyMapFishApp sqlalchemy=true

* Εγκαθιστούμε τις JavaScript librairies στην εφαρμογή:

    $ paster create -t mapfish_client MyMapFishApp

Οι προαναφερθείσες εντολές θα πρέπει να έχουν δημιουργήσει την εφαρμογή MapFish στον κατάλογο ``MyMapFishApp`` .

.. Σημείωση:

    Εάν η δοκιμαστική εφαρμογή τρέχει (από το `Run the minimal application`_ section)
    τότε το port 5000 χρησιμοποιείται ήδη, και θα πρέπει να συντάξετε το 
    ``MyMapFishApp/development.ini`` αρχείο και να αλλάξετε το port number από 5000
    για το παράδειγμα σε  5001.

Στο σημείο αυτό θα πρέπει να μπορείτε να τρέξετε την εφαρμογή με:

    $ cd MyMapFishApp
    $ paster serve development.ini

Ανοίξτε http://localhost:5000 (ή http://localhost:5001) στον browser σας. Θα πρέπει να βλέπετε την ίδια μικρότερη εφαρμογή, όπως αυτή παρουσιάστηκε σε προηγούμενη ενότητα.

Το πλαίσιο εργασίας τουMapFish προσφέρει περισσότερα από αυτά. Συγκεκριμένα, επιτρέπει την εύκολη και ξεκούραστη παροχή υπηρεσιών δικτύου για δημιουργία, ανάγνωση, αναβάθμιση και διαγραφή χαρακτηριστικών.  Παρακαλώ επισκευθείτε το  http://www.mapfish.org για να μάθετε περισσότερα για το MapFish.

Εγκατάσταση του MapFish
===============

Όως υποδείχθηκε και σε προηγούμενη ενότητα, το πλαίσιο εργασίας του MapFish είναι εγκατεστημένο στο live DVD στο virtual Python περιβάλλον, το οποίο βρίσκεται στο ``/usr/lib/mapfish/mapfish-venv`` directory. Ωστόσο, εξακολουθήστε την ανάγνωση εάν επιθυμείτε να μάθετε παραπάνω στοιχεία σχετικά με την εγκαστάσταση του MapFish σε ένα φρέσκο, καθαρό, εικονικό περιβάλλον.

Η εγκατάσταση του πλαισίου εργασίας του MapFish γίνεται με τη γραμμή εργαλείων:

    $ go-mapfish-framework-all.py --python=python2.5 venv

Η παραπάνω εντολή δημιουργεί το περιβάλλον του virtual Python στον κατάλογο
``venv`` , φέρνει το MapFish και όλα τα εξαρτώμενα από αυτό αρχεία και τα εγκαθιστά στο εικονικό περιβάλλον.

Στη συνέχεια, αφού το πλαίσιο εργασίας έχει εγκατασταθεί, μπορεί κανείς να πάει πίσω στην ενότητα `Create a MapFish
application`_ section και να δημιουργήσει μια εφαρμογή MapFish με χρήση αυτή της εγκατάστασης.
